Git

  • git status-показывает состояния файлов в рабочем каталоге и индексе: какие файлы изменены, но не добавлены в индекс; какие ожидают коммита в индексе. Вдобавок к этому выводятся подсказки о том, как изменить состояние файлов.
  • git init-инициализирует репозиторий Git в выранной папке .
  • git add- выбирает конкретный файл для включения в будущий коммит, добавляя его в индекс
  • git commit-m-формирует новый коммит с комментарием, описывающим суть внесенных правок.
  • git push- передает сделанные локольно изменения в связанный удаленный репозиторий для их сохранения и совместной работы
